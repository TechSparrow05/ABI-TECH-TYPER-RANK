📄 ABI TECH - TYPER RANK: Typing Speed Game
────────────────────────────────────────────

A futuristic typing speed test web application with ranking stats, paragraph mode, theme toggle, and animated star backgrounds — built and branded by ABI TECH.

────────────────────────────
📁 PROJECT STRUCTURE
────────────────────────────
/index.html           → Main typing game interface
/style.css            → Included via <style> in HTML
/script.js            → Included via <script> in HTML
/assets/
  └── favicon.png     → Tab icon (browser top bar)
  └── logo.png        → ABI TECH corner branding logo
  └── star.png        → Animated star drop background

────────────────────────────
🚀 FEATURES
────────────────────────────
✔ Typing Speed Game with Real-Time Stats:
   • WPM (Words Per Minute)
   • Accuracy %
   • Mistake Counter
   • Countdown Timer

✔ Word Modes:
   • Beginner (easy words)
   • Advanced (hard words)
   • Custom word input
   • Paragraph/Story Mode (500–1000 characters)

✔ Themes:
   • Light / Dark / Default
   • Smooth toggle switch

✔ Typing Feedback:
   • Green = Correct letters
   • Red = Incorrect letters

✔ Background Effects:
   • Toggle animated star fall ✨
   • Neon glowing background
   • Hero background with parallax effect

✔ Logo & Branding:
   • ABI TECH logo on browser tab (favicon)
   • Logo displayed at top-left corner

✔ Full Responsive Design:
   • Mobile, Tablet, and Desktop friendly

────────────────────────────
🖥️ HOW TO RUN
────────────────────────────
1. Open `index.html` in any browser (Chrome recommended)
2. Click "Beginner", "Advanced", or "Paragraph"
3. Start typing in the input box below the words
4. View your WPM, accuracy, and stats in real-time
5. Use buttons to shuffle, restart, or toggle effects

────────────────────────────
📦 CUSTOMIZATION
────────────────────────────
🖼️ To change the background star image:
  Replace `assets/star.png` with your own PNG (transparent preferred)

🖼️ To change the site logo:
  Replace `assets/logo.png` and `favicon.png` with your own branding

🖱️ Add More Paragraphs:
  Edit the `loadParagraphMode()` function in the JS section

🛠️ Advanced Modifications:
  You can connect Firebase or a database to store scores, rankings, and login support.
